# ELC4L

ELC4L (ELBIXPLUGIN) — VST 플러그인 프로젝트

간단 소개
- C++로 작성된 VST 플러그인 소스와 빌드 스크립트를 포함합니다.

빠른 시작(Windows)
1. Visual Studio와 CMake를 설치하세요.
2. 루트에서 `build.bat` 또는 `build_all.bat`을 실행하거나, CMake를 사용해 솔루션을 생성한 뒤 `build/`에 있는 `.sln`을 Visual Studio로 엽니다.

기여
